# PGATS - Programação para automação de testes - Desafio.

## Introdução
Projeto desenvolvido em **JavaScript** e **Mocha** para práticar a **lógica de programação** e as **execuções de teste**, foi utilizado:

- Vetores
- Estruturas condicionais (`if`)
- Estruturas de repetição (`for`)
- Funções
-  Testes automatizados 

## Descrição do problema
 Um usuário de um site, deve realizar um login.
 Declare um vetor contendo informações sobre usuários de um site, contendo as propriedades: id, nome, email, senha e expirado (tipo: boleano, pode ser true ou false). Adicione ao menos um dos usuarios como expirado sendo true.
 
O sistema deve:
- Validar se o email e a senha foram informados e se está no cadastro.
- Buscar usuário pelo email.
- Verificar se a credencial está expirada.
- Validar senha incorreta para usuário. 
- Validar usuário não encontrado.

## Rodando o projeto

* Rode o console na pasta em que você baixou os arquivos;

* Insira o comando npm install;

* Depois, o comando npx mocha para rodar.
